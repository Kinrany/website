Стек:

 - MongoDB
 - Node.js
 - Express
 - Vue.js

Установка:

 - Установить MongoDB
 - Добавить `<path>/MongoDB/Server/<version>/bin` в PATH
 - Склонировать репозиторий

Запуск:

 - Запустить MongoDB с помощью `mongodb_start.bat`
 - Запустить сервер с помощью `node server.js`
 - Проверить, открыв `http://localhost:8080/`
